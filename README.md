# React To-Do List

This project is an exploration of **Test-Driven Development (TDD) with React**. TDD encourages writing cleaner code, improving accessibility, and adopting a structured approach to building React applications.

You can view the current version of the app [here](https://react-todo-list-nine-theta.vercel.app/).

## Design

The app was designed in Figma with a mobile-first approach in mind.  

[View the prototype](https://www.figma.com/proto/6Daph1ljUjujdchrufDBaq/Todo-list?node-id=6-163&starting-point-node-id=6%3A163&t=LnwWv1UKm5x6NutP-1)

## Current Features (In Development)

- Create a task item
- Edit a task item
- Remove a task item
- Marking tasks as completed  
- Responsive design for mobile and desktop  
- Accessibility considerations are being integrated  
- TDD practices using Jest for component and functionality testing  

## Testing & Deployment

- **Testing:** The project is being tested with Jest to ensure components and functionality behave as expected.  
- **Deployment:** The app is deployed on Vercel and updated as development progresses.

### Running Tests

To run tests locally:  

```bash
npm run test
```

### Installation

To install the app locally:

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```
